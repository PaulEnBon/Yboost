Yboost - API Pokémons

Cette page est affichée par GitHub Pages. L’API Express fonctionne en local ou sur un hébergement Node.js.

Endpoints API

  • GET /api/pokemons - Liste des Pokémons
  • GET /api/pokemons/:id - Pokémon par ID
  • GET /api/pokemons/:id/:name - Pokémon par ID + nom

Local

  • Démarrer le serveur : npm start
  • Tester la liste : http://localhost:3000/api/pokemons
